Aplikacja do dbania o rośliny

O projekcie:

Naszym projektem jest aplikacja mobilna pomagająca w dbaniu o hodowane rośliny. Użytkownik po założeniu konta będzie miał możliwość dodawania swoich roślin do listy oraz wyświetlania informacji o nich. Dane będą pobierane za pomocą API https://trefle.io. Aplikacja oferuje też możliwość ustawiania cyklicznych powiadomień o podlewaniu roślin.
